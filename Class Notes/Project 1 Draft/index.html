Bootstrap Modal Cheatsheet

An introduction to building modals in bootstrap

What are Modals

Modals are pop-up windows that can appear when a page loads or when triggered by a button. Modals are used for alert pop-ups, mailing list sign-ups, welcomes messages and log-in prompts.

How to make a Modal

A modal is composed of 3 main parts: Header, Body & Footer. The Header is where the title and close button are placed. In the body there can be text, images or videos. In the footer there should be an action button like "Sign up!" You will need to use HTML, CSS & JavaScript. There are lots of variations of the basic modal and they can be viewed in the Bootstrap 5 documentation.
